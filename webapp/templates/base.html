Movie Recommendation Classifier

This web application is a proof-of-concept sentiment analysis of movie reviews. Uses a Stochastic Gradient classifier from Python Sci-kit. Training is done offline and the pickled classifier copied over to Python Anywhere for online recommendations. Also allows incremental learning by capturing user feedback. Code and documentation at https://github.com/rohitnayak/movie-review-sentiment-analysis
